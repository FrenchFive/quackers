About Quackers

Quackers is an open-source Discord bot written in Python. It provides fun commands like /duck, helpful utilities like /info.

The project includes a web dashboard built with Flask for easy server configuration—from welcome messages and economy settings to mini-games and AI options. All configuration is stored in a SQLite database.

This repository is maintained by FrenchFive. Contributions are welcome on GitHub!
